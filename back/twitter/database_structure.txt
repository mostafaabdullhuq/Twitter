Users
- id (Primary Key)
- firstName (required)
- lastName (required)
(first name + last name = 20 characters)
- username (min 3 max 15 characters) (required, unique)
- password (min 10 max 24 lowercase, uppercase, special character [_ ! @ # $ % ^ *], number) (required)
- email (required, unique)
- profile_picture (5MB jpg, png) (optional)
- cover_picture (5MB jpg, png) (optional)
- bio (optional) (max 100 character)
- location (optional)
- website (optional)
- timestamp (automatic)
- google_access_token (optional)
- facebook_access_token (optional)
- date_of_birth (optional)
- gender
- phone_number (optional , unique)
- email_verified_at (required not null)



Tweets
- id (Primary Key)
- text (optional max 500 character)
- user_id (Foreign Key referencing Users)
- created_at
- updated_at
- schedule_date_time (optional)


Media
- id (Primary Key)
- parent_type (post, reply)
- parent_id (post_id, reply_id)
- media_type (photo, video)
- media_url (path)
- created_at



Retweets
- id (Primary Key)
- text (optional max 300 character)
- tweet_id (Foreign Key referencing Tweets)
- user_id (Foreign Key referencing Users)
- created_at

Notifications
- id (Primary Key)
- user_id (Foreign Key referencing Users)
- type (e.g., like, retweet, follow, mention)
- source_id (Foreign Key referencing the relevant entity, e.g., Tweets, Users)
- is_read (boolean)
- created_at
